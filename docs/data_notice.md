# Data Notice

This repository contains the CMID corpus for research-oriented Chinese medical intent detection experiments.

The original CMID README states that CMID cannot be used for projects other than scientific research. Before publishing this repository publicly, confirm that redistribution is permitted by the dataset owner or replace `data/raw/CMID.json` with download instructions.
